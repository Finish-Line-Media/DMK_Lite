A streamlined WordPress theme for creating professional digital media kits.

== Description ==

MediaKit Lite is the perfect WordPress theme for speakers, authors, podcasters, and entrepreneurs who need a professional online media kit. Showcase your work, books, speaking topics, and media appearances with style.

= Key Features =

* Professional hero section with auto-contrast text
* Books showcase with cover images
* Speaking topics section
* Podcast/show promotion area
* Companies/affiliations display
* Media questions for journalists
* Investor information section
* Media appearances showcase
* Customizable section ordering (drag & drop)
* Auto-contrast color system for accessibility
* Mobile-responsive design
* Social media integration
* Contact/booking information

= Perfect For =

* Professional Speakers
* Authors and Writers
* Podcasters
* Entrepreneurs
* Media Personalities
* Consultants
* Anyone needing a professional media kit

== Installation ==

1. Upload the theme folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to Customize to set up your media kit content
4. Start with Brand Settings to set your colors
5. Add your content to each section
6. Optionally reorder sections in Section Order

== Frequently Asked Questions ==

= How do I set up my media kit? =

After activating the theme, go to Appearance > Customize. Start with Brand Settings to set your colors, then work through each section adding your content.

= Can I reorder the sections? =

Yes! Go to Customize > Section Order and drag the sections to reorder them. The Hero and Bio sections are fixed at the top.

= Does this theme require any plugins? =

No, MediaKit Lite is designed to work without any required plugins. All features are built into the theme.
